ARGUS SPEAK LAB-X — Política de Privacidad

Última actualización: 09/Feb/2026

1. Datos que recopilamos

  • Correo electrónico (para autenticación y acceso).
  • Estado de suscripción (FREE/PRO) y datos técnicos básicos para operación y mejora del servicio.

2. Pagos

Los pagos se procesan mediante Stripe. ARGUS SPEAK LAB-X no almacena información de tarjetas.

3. Uso de IA

Algunas funciones usan servicios de IA para generar respuestas. No ingreses información sensible, confidencial o personal que no quieras compartir.

4. Cómo usamos tus datos

  • Para permitir inicio de sesión y gestionar tu acceso.
  • Para habilitar PRO cuando corresponda.
  • Para soporte y mejora del Servicio.

5. Eliminación y solicitudes
